# Lumen Studio

AI-Powered Creative Platform - Generate images, videos, edit photos with natural language.

## Features

- 🖼️ **Image Generation** - FLUX, SDXL, logos, portraits, products
- 🎬 **Video Production** - 260+ templates (Hollywood, Animation, Anime)
- 📷 **Photo Editing** - Professional retouching, restoration, compositing
- 😂 **Meme Generator** - 130+ meme templates
- 🌍 **Location Swap** - Put photos anywhere in the world
- 💒 **Wedding Photography** - Full editing suite

## Powered By

- DGX Spark (NVIDIA GB10)
- ComfyUI with 365GB+ models
- GPU-accelerated (22 seconds per image)

## Usage

Just describe what you want in plain English:

```
"Create a logo for Lumen AI, modern tech style"
"30-second product ad, Apple keynote style"
"Professional skin retouch, magazine quality"
"Put us in front of Eiffel Tower at sunset"
"Drake meme: manual work vs AI agents"
```
